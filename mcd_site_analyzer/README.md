# 麦当劳选址打分工具

基于高德地图 POI 数据，对候选点位按麦当劳选址逻辑自动打分。

## 选址逻辑

打分维度（满分 100）：

| 维度 | 满分 | 说明 |
|---|:---:|---|
| KFC 已开门店 | 15 | **最强先行指标**：麦当劳县域拓店有跟随 KFC 的特征 |
| 中产消费力（星/瑞/必） | 15 | 星巴克 / 瑞幸 / 必胜客 = 商圈消费力指标 |
| 学校 | 15 | 6–14 岁儿童 + 高中生是麦当劳核心客群 |
| 住宅小区 | 15 | 外卖订单 + 家庭客来源 |
| 商场 / 广场 | 5 | 盒子店落位空间 |
| 影院 | 5 | 周末家庭客流 |
| 医院 | 5 | 工作日刚需流量 |
| 超市 | 5 | 客流叠加 |
| 快餐竞争密度 | 20 | 华莱士 / 塔斯汀 / 汉堡王 = 商圈快餐需求验证 |

**特殊判定**：周边 1km 内已有麦当劳 → 直接淘汰。

## 使用方法

### 1. 申请高德 API Key

- 访问 https://console.amap.com/dev/key/app
- 创建应用 → 添加 Key → 选择"Web 服务"类型
- 个人开发者免费额度：每天 5000 次调用

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置

编辑 `config.py`，把 `AMAP_KEY` 改成你的真实 Key。

### 4. 编辑候选点位

编辑 `locations.csv`，每行一个候选点：

```csv
name,city,keyword,note
铂金湾国光,信丰县,铂金湾国光,信丰住宅楼盘配套商业
```

字段说明：
- `name`：自定义显示名（任意）
- `city`：高德搜索时限定的城市，需写"赣州市XX区"或"XX市/县"
- `keyword`：高德地图里能搜到的关键词（建议先在地图 APP 里搜一次确认能搜到）
- `note`：备注（不影响计算）

### 5. 运行

```bash
python analyzer.py
```

### 6. 看结果

`output/` 目录下：
- `raw_survey.json` — 完整原始 POI 数据
- `scores.csv` — Excel 可直接打开
- `scores.md` — Markdown 表格
