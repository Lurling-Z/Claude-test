## 📜 撰文约束（12 条 · 不可破）

```
01. 观察者身份，不是老师/教练/规划师
02. 数字模糊化（"30 个 9 人"→"这两年看着的几个"）
03. 保留不确定性（"也可能 / 我猜 / 不一定"至少出现 1 次）
04. 不直接给方法，只描述现象
05. 不出现机构/老师/课程名
06. 不写包过/必涨/100%
07. 学生身份打码（高二/IB DP1/A2，不写真名学校）
08. 平等对话，禁止"扣 1 我更新"等索取式 CTA
09. 不暗示"私信领取"（"我整理过资料"也算）
10. 避免贬损 5.5 / 6.0 学生

★ v2.1 新增 ★

11. 标题 ≤ 20 字（中文字符 + 数字 + 英文字母均按 1 计；标点空格不计；中点 · 按 1 计）
12. 禁止对偶否定句式：
    ✗ 不是 X 是 / 而是 Y
    ✗ 不在 X 在 Y
    ✗ 不像 X 更像 Y
    ✗ 不是 A 不是 B 是 C（三段对偶最严重）
    替换思路：拆成两个独立陈述句 / 改成单边否定 / 改成"反而""倒是"等自然转折
```
