## 1. 标题拆解

```
意外  发现  了  雅思写作7.0  的  办法
└悬念┘ └过程┘  └────结果────┘ └承诺┘
```

| 元素 | 作用 | 替代写法对比 |
|---|---|---|
| **意外** | 制造好奇缺口 | "分享一下"（弱）/ "干货"（弱） |
| **发现** | 暗示"非套路"、有独家方法 | "总结"（机构感重） |
| **雅思写作 7.0** | 锚定具体分数（7.0 是雅思留学申请的关键门槛） | "高分"（模糊） |
| **的办法** | 工具属性，引发收藏冲动 | "经历"（情感属性，只引发点赞） |

**核心公式**：
```
[反常副词] + [发现/找到/试出] + [具体数字结果] + [的+方法/思路/模板]
```

## 3. 正文文案

```
终于想起更新雅思了哈哈哈哈哈，
实在抱歉各位，
心血来潮写了写，
希望有用~
```

四句话三个动作：
1. **拖更道歉** → "更新" = 暗示有粉丝在等 = 社会证明
2. **去精致化** → "心血来潮"反营销，对冲机构嫌疑
3. **低姿态承诺** → "希望有用" 不替读者下判断

**对照写法（机构绝对不能用的）**：
> ❌ "干货 | 雅思写作 7.0 备考全攻略，建议收藏！"

- "这条爆了 → 内容应该真不错 → 我也收藏一下"

### 7.1 标题改造（保留公式）

| 原版（考生人设） | 渊学通版（机构人设） |
|---|---|
| 意外发现了雅思写作 7.0 的办法 | 带学员从 5.5→7.0，写作就靠这 5 个动作 |
