通用中文文案撰写规范（Brand-Agnostic）

———

一、平台与字数规则

（1）小红书正文：≤ 1000 字符（含话题、空行、标点）
（2）小红书标题：≤ 20 字，前 2 行决定打开率
（3）公众号长文：不限字数，首屏 200 字内必须有钩子
（4）公众号推文：推荐 800—1500 字
（5）微博：单条 ≤ 140 字
（6）知乎答：3000 字内为佳，关键论点前置，不要"开局先谢邀"
（7）PDF 文档：内部 = 目录 + 章节 + 末页；对外 = 封面 + 正文 + 联系页

———

二、禁用句式（去 AI 味·零容忍）

（1）不是……而是……（含变体：不是……，是……；不只……，更是……）
（2）你以为……其实……
（3）很多人都不知道……
（4）看完这篇就够了
（5）狠狠拿捏
（6）真正懂行的都……
（7）一文搞定 / 一篇看懂 / 收藏不亏

替代思路：用具体场景、画面、数据替代修辞反差。少用"反差感"做钩子。

———

三、AI 味高频词清单（能不用就不用）

（1）副词类
其实 / 本身 / 往往 / 说到底 / 真正的 / 严格意义上 / 从某种意义上 / 不可否认

（2）升华万能句
"这件事比 XX 重要得多"
"更像是……"
"本质上是……"
"说白了就是……"
（每段都升华一次的写法 → 通篇最多 1 次）

（3）工整四件套（结构对称的并列堆叠）
"学习、练习、反馈和作息" / "X、Y、Z 和 W" 这种四件套全文最多 1 处

（4）万能转折
但同时 / 话说回来 / 与此同时 / 不仅如此 / 退一万步说 / 当然了

（5）抽象黑话
赋能 / 闭环 / 抓手 / 价值感 / 落地 / 心智 / 复用 / 颗粒度 / 沉淀

———

四、表达规范（口语化要求）

（1）第一人称用"我 / 我们"，不要做旁观者解说
（2）段落长短刻意不一，避免每段对称
（3）列举优先用顿号、破折号、空行；少用工整排比
（4）多写场景、人物、画面，少抽象总结
（5）加真实细节最好，无真实素材不要硬编
（6）标点：少用书名号、感叹号；问号留给真问题；省略号克制
（7）数字优先于形容词："3 天没睡好"比"非常疲惫"有力

———

五、对外 / 对内分级

（1）对外（用户 / 客户 / 公众）
去销售话术：不出现"早鸟价""限时""现在抢占名额"等硬广用语
去竞品对比：不主动踩同行
去内部黑话：术语用人话翻译

（2）对内（培训 / SOP / 底册）
销售话术、用户画像、竞品对比、Cross-Selling 可保留
直接的销售逻辑表达可使用

———

六、原始素材清理规则

（1）剔除项
扫码添加 / 原创内容 / 阅读量 / 点赞 / 关注 / 在看 / 转发朋友圈 / 求三连 / 火爆 emoji / 夸张箭头标记

（2）100% 保留项
具体年份、政策、城市、地址、人名、数据、时间节点（可专业升华，不可虚构）

———

七、排版规则

（1）PDF 文档
禁用 Markdown 与 LaTeX
使用大写数字（一、二、三）+ 小括号数字（1）+ 分界线（———）+ 普通空格缩进

（2）小红书
短段、空行多
"———"分隔每篇不超过 3 处
话题统一放尾部，10—12 个为佳，混合"宽词 + 精准搜索词 + 品牌词"

（3）公众号
首屏 3 行内必须钩子
中段每隔 2—3 屏一个分镜或小标题
文末固定栏目

———

八、写完前的快速自查清单

（1）段落长度是否过于均匀？（超过 5 段长度接近 → 重写）
（2）是否出现 3 处以上"其实 / 本身 / 往往"？（→ 删）
（3）是否每段都升华？（→ 全文留 1 次）
（4）是否出现禁用句式？（→ 必改）
（5）能否换成第一人称口吻？（→ 优先换）
（6）是否有具体场景 / 画面？（→ 没有就加 1—2 个）
（7）句尾是否都是"……重要得多 / ……才是关键"？（→ 至少改一半）
