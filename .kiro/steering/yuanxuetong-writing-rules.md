渊学通文案撰写约束规范

———

一、适用范围

本规范适用于渊学通所有文案产出场景：小红书帖、公众号文章、对外客户资料、内部培训手册、销售话术、产品说明书等。

———

二、平台与字数规则

场景 / 字数 / 排版限制

（1）小红书正文：≤ 1000 字符（含话题、空行）
（2）小红书标题：≤ 20 字，前 2 行决定打开率
（3）公众号文章：长文不限，首屏 200 字内必须有钩子
（4）内部 PDF 手册：A4，目录 + 章节 + 末页结构
（5）对外 PDF 资料：A4，封面 + 正文 + 联系页结构

———

三、品牌核心调性

调性：陪伴、结果导向、反鸡血、国际生家庭友好
核心精神（可化用，不必逐字搬）：用环境把人托起来
运营账号：渊学通杭州（小红书认证专业号）
目标用户：国际课程在读生（A-Level / IGCSE / AP / IB）+ 中产及以上家庭，"家长焦虑 + 孩子执行力不足"画像

———

四、禁用句式（去 AI 味·零容忍）

（1）不是……而是……（含变体：不是……，是……）
（2）你以为……其实……
（3）很多人都不知道……
（4）看完这篇就够了
（5）狠狠拿捏

替代思路：用更口语、更具体的描述替代修辞反差。例如把"不是 A，而是 B"改成"A 是有，但更关键的是 B"或直接讲事实。

———

五、AI 味高频词清单（能不用就不用）

（1）副词高频词
其实 / 本身 / 往往 / 说到底 / 真正的 / 严格意义上

（2）升华类
"这件事比 XX 重要得多"
"更像是……"
"本质上是……"
（每段都升华一次的写法 → 通篇最多 1 次）

（3）工整四件套
"学习、练习、反馈和作息"这种排比罗列偶尔用一次即可，不要全文都用

（4）万能转折
但同时 / 话说回来 / 与此同时 / 不仅如此

———

六、表达规范（口语化要求）

（1）第一人称用"我 / 我们"，不要做旁观者解说
（2）段落长短刻意不一，避免每段对称
（3）列举优先用顿号、破折号、空行；少用工整排比
（4）多写场景、人物、画面（"嘴上跟着、心里没跟"），少抽象总结
（5）加真实细节最好（"昨天有个 IB 孩子……"），但无真实素材不要硬编
（6）标点：少用书名号、感叹号；问号留给真问题

———

七、对外 / 对内分级

（1）对外文案（小红书 / 公众号 / 客户资料 / 朋友圈）
去销售话术：不出现"早鸟价""限时""现在报名抢占名额"等硬广用语
去竞品对比：不主动踩同行
去内部黑话：4V1、督学打卡、专属档案等术语用人话翻译

（2）对内文案（顾问培训手册 / SOP / 销售底册）
销售话术、家长心理画像、竞品对比、Cross-Selling 全部保留
可使用直接的销售逻辑表达

———

八、原始素材清理规则

（1）自动剔除：扫码添加 / 原创内容 / 阅读量 / 点赞 / 关注 / 在看 / 转发朋友圈等运营废话
（2）100% 保留：具体年份、政策、城市、校区地址、老师姓名、课时数据、特惠节点
（3）可专业升华，不可虚构数据或老师背景

———

九、排版规则

（1）内部 PDF 手册
禁用 Markdown 与 LaTeX
使用大写数字（一、二、三）+ 小括号数字（1）+ 分界线（———）+ 普通空格缩进

（2）小红书帖
短段、空行多
"———"分隔可用，每篇不超过 3 处
话题统一放尾部，10—12 个为佳，混合"宽词 + 精准搜索词 + 品牌词"

（3）公众号文章
首屏 3 行内必须钩子
中段每隔 2—3 屏一个分镜或小标题
文末固定栏目：报名通道 / 联系方式 / 校区地址

———
