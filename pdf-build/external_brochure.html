!为什么是封闭营，而不是补习班？

普通补习班解决的是"知识缺口"，封闭营解决的是"环境缺口"。当一个孩子从早到晚被放在一个只有学习、只有同伴、只有老师陪着的物理空间里，他不需要靠意志力对抗手机和惰性——环境本身就替他完成了大部分自律。

这就是我们坚持把封闭营办在国际学校真实校园场景中的原因：完整的教室、自习室、图书馆、练习室和宿舍区，让"学习"成为这15天里唯一自然的事。

03

四大班型 · 找到属于孩子的那一档

FOUR-TIER PRECISION TEACHING

不同基础的学员需要不同节奏的课程。我们将所有学员入营前进行精准雅思水平测试，根据真实能力分入对应班型，避免"5.5分的孩子和7分的孩子坐在同一间教室"的低效混班。

入门班零起点 / 初中英语水平
核心模块：词汇基础夯实 · 语法体系搭建 · 雅思题型初识
教学重点：从听说读写最基础的能力维度搭建框架，帮助学员从"陌生"到"熟悉"
基础班高中英语 / 目标5.0—6.0分
核心模块：四项能力分项打磨 · 基础题型解题逻辑 · 应试节奏建立
教学重点：从"听懂、看懂"过渡到"答对"，建立稳定的应试反应
强化班已有基础 / 目标6.0—6.5分
核心模块：题型技巧深化 · 写作结构与逻辑 · 口语Part 2与Part 3拓展
教学重点：从"答对"升级到"答得高分"，攻克写作与口语两大瓶颈
冲刺班目标6.5分以上 / 临考冲分
核心模块：高频考点精讲 · 模考冲刺 · 写作高分句式与口语高分话题
教学重点：从"高分"逼近"目标分"的临门一脚，针对性打磨剩余薄弱点

口语陪练 —— 一对一或小组陪练，覆盖Part 1—Part 3全题型

04

4V1服务天团 · 一个孩子身上有四双眼睛

FOUR-TO-ONE COMPANION SYSTEM

我们认为，封闭营的真正价值不在"上课"，而在"陪伴"。每一位入营学员都拥有由四个专业角色组成的服务团队，从教学、管理、助学到生活全维度托管。

01授课老师
主责教学输出，按班型分项授课。负责听说读写四项的核心讲授、题型技巧拆解、考点路径梳理。
02班主任
主责学习节奏与日程管理，是孩子在营期间的"管家"，定期与家长沟通学员状态与情绪状况。
03助教老师
主责课后答疑、作业批改、督学打卡。确保孩子当天的疑问当天解决，学习行为可见可追踪。
04生活老师
主责食宿安排、情绪疏导、健康管理。让孩子在高强度学习节奏中依然吃得好、睡得好、状态稳。

入营服务流程闭环

第一步 / 入营前精准摸底
入营前进行精准雅思水平测试，全面评估学员能力短板，量身定制专属学习计划。
第二步 / 科学分班
根据测试结果分入入门 / 基础 / 强化 / 冲刺四档班型，确保教学浓度与学员匹配。
第三步 / 专属学习档案
为每位学员建立独立学习档案，写作、口语、模考全程留痕，进步可视化。
第四步 / 家长定期沟通
班主任定期向家长同步学员学习状态、情绪状况、阶段反馈，让家长不再"等结果"。
第五步 / 阶段性模考与动态调整
阶段性评估模考及时跟踪反馈，动态调整教学策略，确保15天每一天都在朝目标分推进。

!关于"无手机管理"

入营第一天起，所有学员手机统一交由生活老师集中保管，每日固定时段发还用于与家长联系。这并非"剥夺"，而是为孩子真正屏蔽掉15天里最大的干扰源——让"专注"成为这个夏天唯一的默认状态。

★阶段性模考与反馈

15天封闭期内安排多轮阶段性评估模考，配合写作精批与口语陪练记录，让每位学员在结营时都能拿到一份完整的进步轨迹与后续学习建议，家长可视化获取孩子真实成长。
